在K8s上部署应用程序

JP 2018-12-15 k8s

创建一个Namespace

k create -f resources/ns.yaml
1
apiVersion: v1
kind: Namespace
metadata:
  name: tutorial
1
2
3
4

创建成功后查看

k describe ns tutorial
1

创建一个Deployment

k apply -f resources/deployment_nginx.yaml -n tutorial
1
apiVersion: apps/v1
kind: Deployment
metadata:
  name: nginx
  labels:
    app: nginx
spec:
  replicas: 1
  selector:
    matchLabels:
      app: nginx
  template:
    metadata:
      labels:
        app: nginx
    spec:
      containers:
        - name: nginx
          image: cargo.caicloud.io/caicloud/nginx:1.9.7
          ports:
            - containerPort: 80

创建后查看是否成功

k get pod -n tutorial -o wide --watch
1

查看Pod的日志

k log <pod_name> -n tutorial
1

在Pod里面执行命令

k exec <pod_name> -n tutorial -- ls -l
1

启动Pod里面的shell

k exec -it <pod_name> -n tutorial bash
1

为Pod添加Label

k label pod <pod_name> ver=v1 -n tutorial
1

查询资源的Label

k describe pod <pod_name> -n tutorial
1

水平扩容

k scale deployments nginx --replicas=4 -n tutorial
1

水平缩容

k scale deployments nginx --replicas=2 -n tutorial
1

通过NodePort暴露服务

k expose deployment nginx --port 80 --type NodePort -n tutorial
1

查看Service

k describe svc nginx -n tutorial
1

查询部署状态

k rollout status deployment nginx -n tutorial
1

部署回滚

k rollout undo deployment nginx -n tutorial
1

查看资源的Yaml文件

k get pod <pod_name> -n tutorial -o yaml
k get svc nginx -n tutorial -o yaml
1
2

删除Deployment

k delete deployments nginx -n tutorial
1

查询删除是否成功

k get all -n tutorial
1
